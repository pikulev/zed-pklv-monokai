# PKLV Monokai

**Classic Monokai with a warmer touch, tailored for Zed.**

![Theme Preview](preview.png)

## Description

This theme represents a port of the **Monokai** color palette to the [Zed](https://zed.dev) editor. 

## Installation

1. Open Zed.
2. Open the **Extensions** view (`Cmd+Shift+X`).
3. Search for `PKLV Monokai`.
4. Click **Install**
5. Select the theme using the theme selector (`Cmd+K` `Cmd+T`).
